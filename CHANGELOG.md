# Changelog

All notable changes to the AletheiaX SDKs are documented here.
This project adheres to [Semantic Versioning](https://semver.org/).

## [0.4.0]
### Added
- `$ALETHIA` holder-tier module in both SDKs (`alethia.myTier` / `alethia.tiers`).
- Python agent-builder toolkit: `Strategy`, `Context`, indicator library, `AgentRunner` (paper mode).

## [0.3.0]
### Added
- Agent marketplace module: list/get agents, measured metrics, equity curve, decision log, allocations.
- TypeScript real-time `WsClient` with auto-reconnect and subscription replay.

## [0.2.0]
### Added
- Trading module: orders, positions, balance, stop-loss / take-profit.
- Wallet-signature authentication.

## [0.1.0]
### Added
- Initial TypeScript + Python clients with public market-data modules.
